================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Cannot load webpage"
Solution: Check web server is running
  cd web_encrypt && python -m http.server 8000

Problem: "ModuleNotFoundError: No module named 'flask'"
Solution: Install server dependencies
  cd server && pip install -r requirements.txt

Problem: "Failed to send to server"
Solution: Check Flask server is running
  cd server && python app.py

Problem: "Cannot decrypt message from server"
Solution: Ensure private key is set on server
  Use POST /set_private_key endpoint first

Problem: Server shows "No public key available"
Solution: Send your public key first
  Click "Send public key to server" button
